# Test Repo 1

This repository is a small test workspace with a couple of sample text files and a secrets warning file. It is useful for checking editor behavior, file handling, and basic documentation changes.

## What’s Included

- `test1.txt` and `test2.txt` as simple sample files
- `secrets.yml` as a reminder not to commit sensitive data
- This README with a short overview of the repo

## Getting Started

There is no build step or application runtime in this repo. Open the files directly in your editor and use them for testing, notes, or quick experiments.
